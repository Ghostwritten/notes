#  Gitbook 插件：页面编辑


---

##  1. edit-link 插件
[edit-link 插件](https://www.npmjs.com/package/gitbook-plugin-edit-link)：添加编辑按钮

###  1.1 安装

```bash
npm i gitbook-plugin-edit-link
```
###  1.2 配置

```bash
    "edit-link": {
     "base": "https://github.com/Ghostwritten/gitbook-docs/edit/master/",
     "label": "Edit"
    }
```
